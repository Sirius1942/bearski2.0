# 领星 ERP 操作场景梳理

## 场景分类

### 1. 产品管理场景

#### 1.1 创建新产品
**业务目标**: 添加新的 Amazon 产品到系统
**操作步骤**:
1. 登录系统
2. 进入产品管理
3. 点击"新建产品"
4. 填写产品信息（SKU、标题、价格等）
5. 上传产品图片
6. 保存产品

#### 1.2 批量导入产品
**业务目标**: 从 Excel 批量导入产品
**操作步骤**:
1. 登录系统
2. 进入产品管理
3. 点击"批量导入"
4. 上传 Excel 文件
5. 确认导入

#### 1.3 修改产品价格
**业务目标**: 调整产品售价
**操作步骤**:
1. 搜索产品
2. 进入产品详情
3. 修改价格
4. 保存

### 2. 订单管理场景

#### 2.1 查询订单
**业务目标**: 查找特定订单
**操作步骤**:
1. 进入订单列表
2. 输入订单号或日期范围
3. 点击搜索
4. 查看订单详情

#### 2.2 批量发货
**业务目标**: 批量处理待发货订单
**操作步骤**:
1. 进入订单列表
2. 筛选"待发货"状态
3. 勾选订单
4. 点击"批量发货"
5. 确认物流信息

#### 2.3 订单退款
**业务目标**: 处理退款申请
**操作步骤**:
1. 进入订单详情
2. 点击"退款"
3. 填写退款原因
4. 确认退款

### 3. 库存管理场景

#### 3.1 查询库存
**业务目标**: 查看产品库存状态
**操作步骤**:
1. 进入仓库管理
2. 选择仓库
3. 搜索 SKU
4. 查看库存数量

#### 3.2 库存调整
**业务目标**: 手动调整库存数量
**操作步骤**:
1. 进入库存管理
2. 搜索 SKU
3. 点击"调整库存"
4. 输入调整数量和原因
5. 保存

#### 3.3 库存盘点
**业务目标**: 定期盘点库存
**操作步骤**:
1. 创建盘点任务
2. 扫描产品
3. 录入实际数量
4. 生成盘点报告

### 4. FBA 管理场景

#### 4.1 创建 FBA 货件
**业务目标**: 向 Amazon FBA 发货
**操作步骤**:
1. 进入 FBA 管理
2. 点击"创建货件"
3. 选择产品和数量
4. 选择目标仓库
5. 生成装箱单

#### 4.2 查询 FBA 库存
**业务目标**: 查看 FBA 仓库库存
**操作步骤**:
1. 进入 FBA 库存
2. 选择站点
3. 查看库存报告

### 5. 财务管理场景

#### 5.1 查看销售报表
**业务目标**: 分析销售数据
**操作步骤**:
1. 进入财务报表
2. 选择日期范围
3. 选择站点
4. 生成报表
5. 导出 Excel

#### 5.2 对账
**业务目标**: 核对 Amazon 账单
**操作步骤**:
1. 进入账单管理
2. 选择账期
3. 查看费用明细
4. 标记已对账

### 6. 广告管理场景

#### 6.1 创建广告活动
**业务目标**: 创建 Amazon 广告
**操作步骤**:
1. 进入广告管理
2. 点击"新建活动"
3. 选择广告类型
4. 设置预算和竞价
5. 选择产品
6. 启动广告

#### 6.2 查看广告数据
**业务目标**: 分析广告效果
**操作步骤**:
1. 进入广告报表
2. 选择日期范围
3. 查看 ACOS、点击率等指标

### 7. 客服管理场景

#### 7.1 回复买家消息
**业务目标**: 处理买家咨询
**操作步骤**:
1. 进入客服邮件
2. 查看未读消息
3. 点击回复
4. 输入回复内容
5. 发送

#### 7.2 处理评论
**业务目标**: 回复产品评论
**操作步骤**:
1. 进入评论管理
2. 筛选待回复评论
3. 点击回复
4. 输入回复内容
5. 提交
